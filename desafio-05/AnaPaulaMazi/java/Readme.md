# Desafio 05 - Java


 **Compilando o programa**

 * Na pasta java abra um *prompt de comando* e adicione:

```
       javac -cp .;lib\gson-2.8.6.jar -d bin ClasseMain.java
```
 **Executando o programa**
 
 * Entre na pasta bin através do comando:
 
 ```
 	cd\bin
 ```
 * Em seguida digite:
 
 ```
 	java -cp .;..\lib\gson-2.8.6.jar ClasseMain <nome-arquivo>
 ```
